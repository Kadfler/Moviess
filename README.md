Movie Catalog — консольный каталог кино

Простое консольное приложение для управления коллекцией фильмов. Добавляйте картины, отмечайте просмотренные, ищите по году выпуска — все данные сохраняются в JSON.

| №   | Возможность            | Что делает                                   |
| --- | ---------------------- | -------------------------------------------- |
|  1  | Показать каталог       | Выводит список всех фильмов: ID, год, статус |
|  2  | Добавить фильм         | Создает запись с новым уникальным ID         |
|  3  | Отметить просмотренный | Меняет статус фильма по его ID               |
|  4️  | Поиск по году          | Находит все фильмы указанного года выпуска   |
|  5  | Выход                  | Сохраняет данные и закрывает программу       |

README.md            # Описание проекта
│

    └── test_logic.py    # Юнит-тесты для функции logic.py

Модуль logic.py

Содержит чистые функции без вывода и взаимодействия с пользователем:

    load_movies(path) — загрузка каталога
    save_movies(path, movies) — сохранение данных
    add_movie(movies, title, year) — добавление фильма
    mark_watched(movies, movie_id) — смена статуса
    find_by_year(movies, year) — поиск по году

Все функции строго возвращают данные по спецификации.
Тестирование (tests/test_logic.py)

Автоматические тесты проверяют:

    Добавление новых фильмов
    Обновление статуса просмотра
    Фильтрацию по году
    Загрузку и сохранение JSON
